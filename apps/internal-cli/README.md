# Internal CLI

Main CLI tool for video production workflows. Run commands with `pnpm cli <command>`.

## Key Commands

- `create-auto-edited-video` - Auto-edit a video with silence removal, bad take detection, and subtitle generation
- `process-queue` - Process videos in the queue
- `article-from-transcript` - Generate an article from a video transcript
- `queue-status` - View current queue status
- `append-video-to-timeline` - Append video to DaVinci Resolve timeline

Run `pnpm cli --help` for all available commands.
