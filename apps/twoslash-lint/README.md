# Twoslash Lint

CLI tool for linting Twoslash code blocks in markdown files. Validates TypeScript code examples for correctness.

## Usage

```bash
twoslash-lint "path/to/files/**/*.md"
```

## Commands

- `pnpm run build` - Build the CLI
- `pnpm run setup` - Link globally for local development
- `pnpm run test` - Run tests
