# @total-typescript/ffmpeg

Video processing toolkit for automated editing workflows. Handles silence removal, bad take detection, subtitle generation, and article creation from transcripts.

## Key Features

- Auto-edited video creation (silence removal, bad take detection)
- AI-powered transcription and subtitle generation
- Article generation from video transcripts
- DaVinci Resolve integration
- Queue-based processing system

Built with Effect-TS for robust error handling and composition.
